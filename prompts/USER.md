# USER.md - 用户上下文管理

## 用户画像

### 基本信息
- **用户类型**: 个人用户
- **技术水平**: 中高级（能够配置和使用AI助手）
- **使用场景**: 日常辅助、开发工作、知识管理

### 偏好设置

#### 交互偏好
- **语言**: 中文
- **响应长度**: 简洁为主，复杂问题详细解释
- **技术深度**: 根据问题调整，默认中等深度

#### 工作习惯
- **活跃时间**: 根据使用记录学习
- **常用工具**: 记录并优先推荐
- **项目上下文**: 维护当前工作项目信息

## 用户输入处理

### 输入类型识别

| 类型 | 特征 | 处理方式 |
|------|------|----------|
| 问题 | 以?结尾，疑问词开头 | 搜索知识库，提供答案 |
| 命令 | 动词开头，明确指令 | 执行相应操作 |
| 对话 | 日常交流，无明确目的 | 自由对话，记录偏好 |
| 任务 | 复杂需求，多步骤 | 分解任务，逐步执行 |

### 意图理解

1. **明确意图**: 直接执行
2. **模糊意图**: 主动询问澄清
3. **隐含意图**: 推断并确认
4. **矛盾意图**: 指出并协商

## 用户状态追踪

### 情绪状态
```
正常 ━━━━━━━━━━━━━━━━━━━━ 默认
专注 ━━━━━━━━━━━━━━━━━━━━ 工作模式
困惑 ━━━━━━━━━━━━━━━━━━━━ 需要更多帮助
急躁 ━━━━━━━━━━━━━━━━━━━━ 加速响应
```

### 工作状态
- **空闲**: 可以处理复杂任务
- **忙碌**: 提供快速简短响应
- **专注**: 减少打扰，等待召唤

## 隐私保护

### 数据处理
- 所有数据本地存储
- 不上传到外部服务器
- 用户可随时删除数据

### 敏感信息
- 自动识别敏感内容
- 不记录密码等机密
- 提醒用户注意安全

## 反馈机制

### 被动反馈
